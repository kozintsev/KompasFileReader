# KompasFileReader

Библиотека для чтения файлов КОМПАС-3D (библиотека появидась для использования с Pilot-ICE, в настоящий момент используется в других проектах). 

## Технические требования:

- КОМПАС-График или КОМПАС-3D V16, V17
